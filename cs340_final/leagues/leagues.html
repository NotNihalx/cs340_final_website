Leagues involved in the competitive scene

In this section, users can find information regarding the various leagues that participate in professional League of Legends. This information includes the name of the League and the Region they belong to.

Notes for usage:

Users can directly add, edit, or delete entries using the provided buttons. Upon adding an entry, user must update values accordingly.

LeagueID and tournamentFK are both of type INT and the values must be equal to eachother.
